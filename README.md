# The Modern Store (API Edition)

A modern webstore project built using [Next.js](https://nextjs.org/), [TailwindCSS ](https://tailwindcss.com/) and [Platzi Fake Store API](https://fakeapi.platzi.com/en)

## Running the project

```bash
git clone https://github.com/serfoll/lexicon-ovning-modern-store

cd lexicon-ovning-modern-store

npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
